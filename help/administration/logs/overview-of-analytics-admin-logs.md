# Présentation des journaux d’administration Analytics

Fichiers journaux permettant de savoir quand se connectent les utilisateurs, leur utilisation, l’accès, les suites de rapports et les changements administratifs. Le journal d’administration consigne toutes les modifications effectuées par les administrateurs dans les outils d’administration. Il fournit une passerelle vers des rapports définis par l’utilisateur depuis n’importe lequel des trois journaux.
