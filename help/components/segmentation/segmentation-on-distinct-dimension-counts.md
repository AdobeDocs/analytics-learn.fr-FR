# Segmentation sur des comptes de dimensions distincts {#segmentation-on-distinct-dimension-counts}

La nouvelle [!UICONTROL Comptage distinct] dans le [!UICONTROL Créateur de segments] vous permettent de segmenter en fonction du nombre d’éléments uniques dans n’importe quel [!UICONTROL dimension]. Filtrez les visites où plus de 5 pages ont été consultées ou les accès où plus de 5 produits ont été achetés, c’est-à-dire dans une seule commande.
